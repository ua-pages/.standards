# Інженерні стандарти `ua-pages`

## Основоположні принципи (LLFF)

- **Local First** — проєкт працює локально, без зайвих зовнішніх залежностей.
- **Lightweight First** — мінімум коду, мінімум залежностей, мінімум складності.
- **Fast First** — швидкий запуск, швидка робота, швидка збірка (або її відсутність).
- **Focused First** — кожен проєкт вирішує одну проблему і робить це добре.

## Інженерні стандарти

- Жодних зайвих залежностей.
- Нативна веб-платформа — понад усе.
- Прогресивне покращення.
- Доступність за замовчуванням.
- Семантичний HTML.
- Продуктивність — понад візуальні ефекти.
- Читабельність — понад хитромудрість.
- Простота — понад абстракцію.
- Маленькі перевикористовувані компоненти.
- AI-дружня архітектура.
- Передбачувана структура папок.
- Готовність до продакшену за замовчуванням.

## Філософія коду

- Кожен файл має призначення.
- Кожна залежність має виправдати своє існування.
- Кожна абстракція має прибирати складність, а не додавати її.
- Видаляй код щойно це можливо.
