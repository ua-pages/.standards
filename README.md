# Інженерні стандарти `ua-pages`

## Навіщо існує цей репозиторій

**Ми робимо світові інженерні знання природними для українських розробників.**

Ми не воюємо зі світовими стандартами. Ми їх поважаємо, використовуємо та розвиваємо разом із ними. Git, GitHub, Node.js, Web Components, CI, Pull Request — це спільна міжнародна мова інженерів.

Водночас ми віримо, що **мислити, навчатися, проєктувати архітектуру та передавати знання можна й потрібно українською мовою**.

Саме тому `ua-pages` будує українську інженерну культуру навколо міжнародних стандартів.

Цей репозиторій є канонічним джерелом інженерних принципів організації `ua-pages`. Тут описано нашу філософію, підхід до розробки, структуру репозиторіїв, правила документації, процес роботи з Pull Request, CI та інші стандарти, яких ми дотримуємося в кожному проєкті.

Ми не прагнемо вигадувати власні інструменти там, де вже існують якісні світові рішення.

Ми прагнемо зробити так, щоб **світові інженерні знання стали природною частиною українського професійного середовища**.
